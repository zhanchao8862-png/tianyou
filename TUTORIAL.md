# 天佑战团源码编辑器 — 使用教程

## 一、快速开始

### 1.1 启动
双击 `tianyou_editor.exe`，程序自动检测上次打开的 MOD 目录并加载。

### 1.2 首次使用
如果还没有 MOD，点击菜单 **文件 → 打开 MOD 目录**，选择骑砍 Module System 的根目录（包含 `module_troops.py`、`module_items.py` 等）。程序会自动：
- 解析所有兵种数据（`module_troops.py`）
- 加载物品列表（`module_items.py`）
- 匹配汉化文件（`troops.csv` / `items.csv`）

---

## 二、兵种列表（左侧面板）

| 操作 | 方法 |
|------|------|
| 选择兵种 | 单击兵种条目 |
| 搜索兵种 | 在顶部搜索框输入，支持 ID 和中文名模糊匹配 |
| 新增兵种 | 点击「新增兵种」按钮 |
| 复制兵种 | 选中后点击「复制兵种」 |
| 删除兵种 | 选中后点击「删除兵种」 |
| 排序 | ▲ / ▼ 按钮上下移动 |
| 保存 | 点击绿色「保存」按钮或按 Ctrl+S |

---

## 三、兵种详情（右侧面板）

### 3.1 三个标签页
- **基本信息**：只读概览，显示所有字段值
- **兵种详情**：结构化编辑面板（见下方）
- **源码**：直接编辑 Python 源码，Ctrl+Z/Y 支持文本级撤销

### 3.2 详情面板字段

| 区域 | 内容 |
|------|------|
| 基础信息 | ID、名称、复数名、阵营（下拉选择） |
| 属性 | 力量/敏捷/智力/魅力/等级，Spinbox 直接改 |
| 熟练度 | 7 项武器熟练度 |
| 标记位 | 20 项 Flags，分组 UI，一键勾选 |
| 技能 | 42 技能，两列 Spinbox，直接改数值 |
| 升级树 | 可搜索下拉框，选择父/子兵种 |

### 3.3 按钮
- **应用修改到当前**：将详情面板修改暂存到内存（可撤销）
- **还原**：从文件重新解析，放弃修改

---

## 四、装备管理

### 4.1 拥有物品（右上角）
- 显示当前兵种携带的所有物品
- **添加**：弹出物品选择对话框，支持搜索、多选、双击移除
- **移除**：选中后删除
- **清空**：全部清空
- 拖拽底部边框可调整面板高度
- 鼠标悬停时滚轮滚动列表

### 4.2 装备槽位（详情面板 - F7）
- 按部位分类显示（头部/身体/腿部/手部/武器/马匹/马甲）
- 双击装备条目可跳转到对应物品

---

## 五、撤销/重做

| 操作 | 快捷键 |
|------|--------|
| 撤销 | Ctrl+Z |
| 重做 | Ctrl+Y |

- 兵种级操作（新增/删除/排序等）使用快照撤销
- 源码编辑使用文本级撤销
- 撤销栈最多 50 步

## 九、常见问题

**Q: 兵种列表显示空白？**
A: 检查 MOD 目录下是否有 `module_troops.py` 文件。

**Q: 保存后游戏里没变化？**
A: 保存仅修改源码文件，需要重新编译模块才生效。

**Q: 汉化不显示？**
A: 确保 MOD 目录下有 `troops.csv` 或 `game_strings.csv` 汉化文件。
